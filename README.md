# EBYS Workflow Automation

A production-grade Chrome extension that automates repetitive document workflows
within the EBYS web application.

## Features
- User-initiated workflow automation
- DOM-based interaction with EBYS UI
- Safe handling of PrimeFaces AJAX lifecycle
- Start / Stop controls with configurable limits
- No data collection or external communication

## Technology
- JavaScript (ES6+)
- Chrome Extensions (Manifest V3)
- DOM automation
- MAIN world and isolated world handling

## Security & Privacy
- Runs only after explicit user action
- Operates within the user's authenticated session
- Stores only local preferences
- Does not collect or transmit any data
